Shticker Book Rewritten

Custom launcher and tool for the Toontown Rewritten MMORPG

Download this project as a .zip file Download this project as a tar.gz file

Shticker Book Rewritten is an all-in-one tool to provide easy access to various resources that are useful when playing Toontown Rewritten. This program is Free and Open Source under the GPL v3 (or higher) and is provided by a third party that is not affiliated with the Toontown Rewritten project in any way.

Shticker Book Rewritten is cross platform and supports both Windows and Linux operating systems. MacOS support is coming soon.

Features

  • Game launcher to log you in and start the game.
  • Automatic downloading and patching of game files.
  • Built in group tracker tab provided by www.toonhq.org.
  • Built in invasion tracker provided by www.toonhq.org.
  • Built in fishing guide and tracker provided by siggen.toontown-click.de.
  • Multi-toon launcher: allows an unlimited number of accounts to be launched all from the same launcher, no need to open a new launcher for each account.
  • Optional credential savings allowing you to login easily without having to remember and type in your username and password. Allows saving multiple accounts at a time.
